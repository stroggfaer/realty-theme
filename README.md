# realty-theme

## О проекте

**Realty Theme** — WordPress-тема для сайта аренды и продажи недвижимости с интерактивным поиском, фильтрацией и системой бронирования объектов.

Проект представляет собой полнофункциональный каталог недвижимости с ролевой моделью (хосты/гости), личным кабинетом пользователя и интеграцией карт Google Maps. Фронтенд построен на Vue.js 3 с компонентным подходом и реактивным управлением состоянием через Vuex.

### Ключевые функции

- 🏠 **Каталог недвижимости** — Custom Post Type `property` с расширенными характеристиками, таксономиями локаций и типов жилья
- 🔍 **Умный поиск** — многопараметрическая фильтрация объектов через AJAX с плагина Anastasia Filter
- 📅 **Система бронирования** — календарь доступности, выбор дат, модальные окна подтверждения
- 👤 **Личный кабинет** — разделение ролей хостов и гостей, управление объектами и бронированиями
- 🗺️ **Интерактивные карты** — отображение объектов на Google Maps с кластеризацией
- 📱 **Адаптивный дизайн** — mobile-first подход, шрифт Manrope, поддержка всех современных устройств

### Архитектура

- **Backend:** WordPress + PHP (Pod Framework для CPT)
- **Frontend:** Vue.js 3 + Element Plus + Vuex
- **Сборка:** Webpack + SCSS
- **Локальная разработка:** http://realproperty.local/

WordPress тема для сайта недвижимости с поддержкой webpack и SCSS.

## Используемые технологии

### Backend/CMS
- **WordPress** — CMS система управления контентом
- **Pods** — плагин для создания кастомных типов записей (CPT) и таксономий

### Frontend инструменты
- **Webpack** — сборщик модулей для JS и SCSS
- **SCSS/SASS** — препроцессор CSS
- **Babel** — транспиляция JavaScript (ES6+)
- **PostCSS + Autoprefixer** — трансформация CSS
- **CSSnano** — минификация CSS

### JavaScript фреймворки и библиотеки
- **Vue.js 3** — прогрессивный JavaScript фреймворк
- **Vuex** — библиотека для управления состоянием
- **Element Plus** — UI-компоненты для Vue 3
- **jQuery** — для совместимости
- **Moment.js** — работа с датами
- **DateRangePicker** — виджет выбора диапазона дат

### Сторонние сервисы
- **Google Maps API** — интерактивные карты
- **Font Awesome** — иконографическая библиотека
- **Splide** — слайдер для галерей

## Структура проекта

```
realty-theme/
├── assets/
│   ├── sass/           # SCSS файлы стилей
│   ├── vue_module/     # Vue.js компоненты
│   ├── images/         # Изображения и иконки
│   └── font-awesome/   # Иконки Font Awesome
├── js/                 # JavaScript файлы
├── inc/
│   ├── admin/          # Админ-функции, регистрация Pods, настройки

└── webpack.vue.config.js # Конфигурация Webpack для Vue
```

## Возможности

- Кастомный тип записи «property» для объектов недвижимости
- Фильтрация объектов по параметрам (ajax)
- Интерактивные карты Google Maps
- Галерея изображений объектов
- AJAX пагинация
- Адаптивный дизайн
- Поддержка русского языка

## Установка зависимостей

```bash
npm install
```

## Скрипты

- `npm run sass` — компиляция SCSS с отслеживанием изменений
- `npm run build:css` — production сборка CSS
- `npm run watch:css` — dev сборка CSS с отслеживанием
- `npm run vue-build` — production сборка Vue компонентов
- `npm run vue-watch` — dev сборка Vue с отслеживанием
